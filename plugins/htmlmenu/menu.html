3. Two-Way Data Binding

Enable "Two-Way Binding" in properties. An input's id must match a C2 global variable name.

4. GSAP Animations

Use GSAP actions to animate elements. Make sure gsap.min.js is in your project files!

5. Focus & Keyboard

Manage focus and keyboard input. Use the buttons in the Flexbox section below to test focus actions.

When typing in an input, IsInputActive is true, and keyboard events won't affect C2.
